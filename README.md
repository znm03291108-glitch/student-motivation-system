# 星星成长学习激励系统

这是一个适合中小学生提高学习积极性的前端项目。

## 功能
- 学习任务打卡
- 积分奖励机制
- 惩罚扣分机制
- 每日签到
- 成长等级
- 学生排行榜
- 家长/老师后台添加任务、奖励、惩罚
- 手机浏览器可直接运行

## 部署方式

### 方式一：直接本地打开
点击 `index.html` 即可运行。

### 方式二：部署到 Vercel / Netlify / Cloudflare Pages
上传整个文件夹即可。入口文件是 `index.html`。

## 文件说明
- `index.html` 页面结构
- `style.css` 页面样式
- `app.js` 系统逻辑
- `README.md` 项目说明
