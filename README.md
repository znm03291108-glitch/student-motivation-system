# 星星成长计划 V3：苏教版一至六年级课程学习版

这是一个适合中小学生学习效率提升的登录版系统，支持 Firebase 登录、云端数据保存、积分奖励、家长/老师监督，以及一至六年级课程学习模块。

## 新增功能

- 课程学习：一年级到六年级
- 学科覆盖：语文、数学、英语
- 册别：上册、下册
- 原创知识点讲解
- 原创例题和同步练习
- 闯关测验自动加分
- 答错自动进入错题本
- 错题复习加分
- 学习进度云端保存
- 家长/老师后台

## 版权说明

本项目不内置教材原文、教材图片或完整课文。课程内容采用原创知识点讲解、学习方法、原创例题和练习题，适合做同步学习辅助系统。

## 部署方式

把以下文件上传覆盖 GitHub 仓库原文件：

- index.html
- style.css
- app.js
- README.md

GitHub Pages 会自动更新。

## Firebase

app.js 已经填入 Firebase 配置。请确保 Firebase 已开启：

- Authentication → Email/Password
- Cloud Firestore → 测试模式或合适安全规则
