# 星星成长计划 V4：课程库 + 题库 + 试卷中心版

本版本在 V3 基础上增加：

- 一至六年级课程库
- 语文、数学、英语
- 上册、下册
- 原创知识点讲解
- 原创同步练习题
- 自动答题判分
- 答对加分
- 答错进入错题本
- 错题复习加分
- 试卷中心：单元卷、期中卷、期末卷、专项训练
- Firebase 登录和云端保存

## 版权说明

本系统没有内置苏教版教材原文、教材图片、完整课文扫描件或第三方试卷原文。内置内容为原创学习讲解、原创例题和原创练习。

如果你拥有授权教材、校本资料、公开授权题库，可以继续导入到系统中。

## 上传方式

把以下文件上传覆盖 GitHub 仓库原文件：

- index.html
- style.css
- app.js
- README.md

然后等待 GitHub Pages 自动部署。
